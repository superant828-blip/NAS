# 2026-03-29 工作日志

## 修复的问题

### 1. 分享API 404问题
- **问题**: 前端调用 `/api/v1/shares/all` 返回404
- **原因**: 后端路由注册问题（路由顺序冲突）
- **解决**: 修改前端调用 `/api/v1/shares/smb/all`（实际工作的API）
- **文件**: ui/js/app.js, ui/mobile.html

### 2. 移动端下载错误
- **问题**: 下载文件只显示35B，实际是错误JSON
- **原因**: 文件路径不存在，返回404错误，前端当作文件下载
- **解决**: 添加错误检测，小文件且包含"detail"时提示"文件不存在"
- **文件**: ui/mobile.html, ui/js/app.js

### 3. 配置保存405错误
- **问题**: PUT /api/v1/config 返回405 Method Not Allowed
- **原因**: FastAPI路由注册问题，main.py中定义的路由无法匹配
- **解决**: 创建独立插件 api/plugins/config.py，将配置路由移至插件系统
- **文件**: api/plugins/config.py, api/plugins/__init__.py

### 4. 前端修改
- app.js: loadShares() 调用 /shares/smb/all
- app.js: downloadFile() 添加错误检测
- app.js: saveAllowedExtensions() 改为 POST /config/update
- mobile.html: loadShares() 调用 /shares/smb/all
- mobile.html: downloadFile() 添加错误检测
