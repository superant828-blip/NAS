## 股市行情API

### 新浪行情API
```
https://hq.sinajs.cn/list=sh688256,sh000688,sh000001
```
- 需要 `Referer: https://finance.sina.com.cn` 头
- 返回 GBK 编码，需 iconv-lite 解码
- 代码格式：`sh{代码}` (上海) / `sz{代码}` (深圳)
- 数据字段：名称、开盘价、昨收、现价、最高、最低、成交量、成交额等

## 采集规则

1. **P0 必采**：每个关键词至少访问 36氪 + 华尔街见闻
2. **P1 选采**：发改委、国家数据局、集微网、生物谷（按话题选择）
3. **跳过策略**：打不开或无结果就跳过，不要卡住
4. **记录格式**：每条资讯 = 标题 + 来源 + 日期 + 摘要
5. **输出文件**：`pipeline/data/collected.json`
