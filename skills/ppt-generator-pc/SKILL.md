# PC版PPT生成器

将讲稿转换为专业的横向(16:9)HTML演示稿。

## 设计哲学

- **一屏一主题** - 每页只讲一个核心观点
- **16:9横向** - 符合PC/投影仪播放标准
- **深色专业风** - 海尔蓝(#0066cc) + 橙色(#ff6600) 主题色
- **视觉层次** - 标题→副标题→内容→数据

## 生成流程

### Step 1: 读取讲稿内容
读取用户原始讲稿，保留核心内容。

### Step 2: 提取关键数据
整理所有数据点：营收3023亿、13座灯塔工厂、50%库存周转提升等。

### Step 3: 设计幻灯片结构
规划页面顺序，确保逻辑流畅。

### Step 4: 生成HTML
生成单个可独立运行的HTML文件，支持键盘/鼠标翻页。

## 视觉规范

| 项目 | 规范 |
|------|------|
| 比例 | 16:9 横向 |
| 最小宽度 | 1200px |
| 背景 | 渐变深色 (#0a0f1a → #1a1a2e) |
| 主色调 | 海尔蓝 #0066cc |
| 强调色 | 橙色 #ff6600 |
| 标题 | 48-64px, 白色 |
| 正文 | 24-32px, 浅灰色 #e0e0e0 |
| 数据 | 橙色高亮 #ff6600 |

## 交互要求

- 键盘 ← → 翻页
- 鼠标点击左右区域翻页
- 底部页码指示器
- 平滑过渡动画 (0.5s)

## 输出

生成单个 HTML 文件，保存在用户指定路径。
